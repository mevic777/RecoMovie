Pentru a porni aplicatia, trebuie de instalat versiunea Python 3.14 pe sistemul de operare, iar apoi de rulat comandat urmatoare: py -m venv .venv, pentru a crea 
variabila locala pentru proiectul dat. Dupa acest moment, trebuie de rulat comanda: pip install -r requirements.txt, pentru a instala toate pachetele de care are
nevoie aplicatia pentru a rula cum trebuie. Dupa acest pas, ca sa rulati aplicatia in sine, este nevoie sa puneti urmatoarea comanda: py app\main.py.
